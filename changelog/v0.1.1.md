# v0.1.1 — 2026-07-07

First published release (npm + official MCP Registry as
`io.github.LizzleChen/local-dfw-mcp`). v0.1.0 was never published — the MCP
Registry namespace is case-sensitive to the GitHub username and validates
against the immutable npm tarball, so the casing fix forced a bump.

## Added

- Nine tools: `about`, `dfw_health`, `dfw_311`, `dfw_crime` (City of Dallas),
  `dfw_fema_flood`, `dfw_tea_schools`, `dfw_nws_alerts`,
  `dfw_utility_providers`, `dfw_district_lookup`.
- Core plumbing (`lib/`) ported from local-austin-mcp (Apache-2.0, attributed
  in NOTICE): SODA/ArcGIS adapters, cache, retry profiles, semaphore,
  registration wrapper.
- **Metro router with jurisdiction pre-flight.** Postal "Dallas, TX" is not
  City of Dallas jurisdiction: city-scoped tools verify the address
  (ZIP/keyword → geocode → city-limits polygon) and refuse with an explicit
  "not covered" message rather than returning plausible-looking results from
  the wrong city's data.
- Verified-source registry (`lib/sources.js`): every dataset is live-checked
  before being wired, and tools refuse to run against unverified entries.
