# v0.2.1 — 2026-07-08

## Added

- `dfw_appraisal`: county appraisal record for an address — owner, land /
  improvement / market value (2025 certified roll), year built, land use,
  acreage — for all four core counties.

## Why it's shaped this way

- Source is **TxGIO StratMap statewide land parcels** (keyless state
  republication of county CAD rolls), not the DCAD/TAD portals, which offer
  no usable public API.
- The service's `query` operation is token-gated; only `identify`
  (point-in-polygon) is public. Hence the tool is geocode-then-identify with
  **no owner-name or free-text search**.
- Census geocodes interpolate into the street right-of-way, so identify uses
  a tuned ~9 m reach (tolerance 8 px + extent pad). Consequence: a single
  match can be the *neighbor's* parcel — output (and the input-field
  description) says to verify the returned situs address.
- Some parcels publish market value "0"; rendered as "value unavailable",
  never $0. Appraised value is not a tax bill; not a consumer report.
